# 🐺 Wild Wolf – The King of Jungle

> AAAAA-Grade VR Strategy-Action Game | Wrap AI Studio

## Overview
Wild Wolf is a massively multiplayer VR strategy-action game built on Unreal Engine 5.6.
500-player boss raids, dual Combat + Strategy modes, 11 mythologies, and FENRIR THE ETERNAL —
a Level 15 boss with infinite HP that can never be defeated. Nature always wins.
